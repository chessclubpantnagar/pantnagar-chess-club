# pantnagar-chess-club
Chess Club Pantnagar is a website created for the chess enthusiasts and hobbyists in GBPUAT, Pant Nagar. 



----------------------------------------- GUIDELINES FOR PRODUCTION-----------------------------------------------

For setup RUN:

    git clone https://github.com/chessclubpantnagar/pantnagar-chess-club.git
    cd pantnagar-chess-club
    npm install

For preview:

    npm start

To push code:

    git add .
    git commit -m "SOME MESSAGE"
    git push origin main

Every Time BEFORE you start to code:

    git pull origin main

To update the live website: 

    npm run deploy
